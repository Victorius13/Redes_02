Modelo OSI (Open Systems Interconnection) - Capas 2-7

Capas del Modelo OSI (Niveles 2-7)

Capa 2:Enlace de Datos

Esta capa establece una comunicación confiable entre dispositivos directamente conectados. Se encarga de organizar los datos en tramas y asegura la detección y corrección de errores en el enlace físico.

  • Define cómo los dispositivos usan direcciones físicas (MAC).
  • Gestiona el control de errores y el control de flujo para evitar colisiones.
  • Actúa como intermediario entre la Capa Física y la Capa de Red.

Capa 3: Red:

La Capa de Red se centra en el enrutamiento de datos entre redes. Utiliza direcciones IP para identificar dispositivos de manera única en una red y selecciona rutas eficientes para transferir datos.

  • Facilita la comunicación entre distintas redes, independientemente del hardware.
  • Opera con protocolos como IP y herramientas como routers.
  • Asegura que los paquetes lleguen al destino correcto, incluso en redes complejas.

Capa 4: Transporte

Esta capa garantiza la transferencia confiable de datos de extremo a extremo. Administra errores, ordena paquetes y ajusta el flujo para evitar congestión en la red.

  • Utiliza protocolos como TCP (confiable) y UDP (menos confiable, pero más rápido).
  • Asegura que los paquetes se entreguen completos y en el orden correcto.
  • Divide los datos en segmentos manejables.

Capa 5: Sesión

Esta capa establece, gestiona y termina sesiones entre dispositivos. Es especialmente útil en aplicaciones que requieren múltiples interacciones entre los mismos puntos finales.

  • Proporciona sincronización y puntos de recuperación en caso de interrupciones.
  • Gestión de autenticación para garantizar conexiones seguras.
  • Útil en servicios como la transmisión de video y la transferencia de archivos.

Capa 6: Presentación

La Capa de Presentación actúa como traductor entre la red y la aplicación, asegurando que los datos sean comprensibles para el sistema receptor.

  • Transforma datos entre formatos, como XML, JSON o binarios.
  • Realiza compresión de datos para mejorar la eficiencia en la transmisión.
  • Incluye funciones de cifrado para garantizar la privacidad y seguridad de los datos.

Capa 7: Aplicación

Es la capa más cercana al usuario y sirve como interfaz para interactuar con la red. Ofrece servicios que permiten la transferencia de datos y la comunicación entre aplicaciones.

  • Proporciona protocolos como HTTP, FTP y SMTP.
  • Facilita el acceso a recursos compartidos como impresoras y servidores.
  • Es clave para aplicaciones como navegadores, correos electrónicos y mensajería instantánea.
